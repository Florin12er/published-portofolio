About me:

Hi, I'm Sebastian, a coding enthusiast with three months of experience. I use JavaScript, CSS, HTML, and TypeScript to create cool web projects. I rely on Git for organization, code with Neovim on Ubuntu, and prioritize efficiency and clean development. I specialize in crafting sleek user interfaces with React and Next.js. Beyond coding, I enjoy weightlifting and reading. Check out my portfolio site for a glimpse into my coding journey.
